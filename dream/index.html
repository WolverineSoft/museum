Compatibility notes. This game is written in Java. You will need a Java Runtime Environment installed to run this game. To run the game, cd to the "dream" folder in your terminal execute the following command: 

java.exe -cp .;jl1.0.jar Game
You do not need to run compile.sh. Confirmed working with the JRE that ships with Minecraft in 2024.
