Mr. T's Freedom Fight (2004)

Authors Pete Knepley (code), Jason Skorski (music)

Description A Wolfenstein-esque 3D shooter where you complete missions according to a story. 

Mr. T's Freedom Fight

This game was written for the Wolverine Soft 48 hour game dev contest in 2004.

Ingredients:
DirectX Graphics API
Bass Sound Lib 2.0
3D Studio MAX 5.1
Goldwave Sound Editor

Features:
3d keyframed models made by Pete
original score by Jason Skorski
Ellipsoid collision detection
Mr. T voiceclips
3D positioned sounds
4 game stages
3 levels of enemy AI
