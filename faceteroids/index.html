Faceteroids (2009)

Authors Jason Robert Eaton

Description 

After extracting the folder, you have to set up the project.  Get to the project tab and find 
the New Project... option.  Browse to this folder, then click the Create Project button.

Then right click the Faceteroids.fla file, and click Make Default Document.

Then you click the Test Project button, and it should create the Faceteroids.swf and run.


For a complete reference to all objects in flash, go to http://livedocs.adobe.com/flash/9.0/ActionScriptLangRefV3/
I used this site to learn nearly everything about specific flash objects, and what it has to offer.  


If you have any trouble with the project, just email me at eatonj@umich.edu and I can help you work it out.
