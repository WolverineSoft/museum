Space Station 13 Virtual Reality Auto Chess (2019)

Authors Tabski, Nabski

Description

A game idea that started out as a meme, and turned into something actually playable (but still a meme).

Space Station 13 Virtual Reality Automatic Chess (or SS13VRAC for "short") is a full auto chess game mode built from scratch over the course of the Wolverine Soft 2019 7 Day Summer Game Jam.

If you can find a friend, VR, and a stable internet connection you'll be able to experience Space Station 13 and Auto Chess like you never have before.
